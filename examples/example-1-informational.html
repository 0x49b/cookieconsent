Informational example

This example informs the user that cookies are used on this website. This is usually 'good enough' to be compliant with the cookie consent law. However, it does not count as 'valid' consent.

Valid consent can only be given if the information regarding the cookie usage on the website is clear and concise, and the user understands exactly what they are consenting to, and the user has an opportunity to refuse and/or revoke consent. Fortunately, these only count for cookies that can personally identify you. (You must still specify all the cookies used on your website in the cookie policy)

Note: If you click a button on the popup, it will permanently dismiss the message. That means that the popup will not be shown again, even after a page refresh. That is unless you clear the previous decision that you made by clicking the 'Clear previous answer' button below (only displayed if an answer has been selected)
