Opt-in example

This example informs the user that cookies are used on this website. In this case, the cookies are disabled by default. Instead of simply informing the user that cookies are used, the user is given prior notice. Therefore cookies are only used if the user explicitly allows it. This is the best way to be compliant with the cookie consent law.

This still only applies for cookies that can personally identify you. (You must still specify all the cookies used on your website in the cookie policy). Therefore any other cookies for styling, functionality or preferences can be used without the need for consent.

Note: If you click a button on the popup, it will permanently dismiss the message. That means that the popup will not be shown again, even after a page refresh. That is unless you clear the previous decision that you made by clicking the 'Clear previous answer' button below (only displayed if an answer has been selected)
